# Microwave Oven

## Description
A Microwave Oven simulation developed using the PIC16F877A microcontroller and Embedded C. The project provides a menu-driven interface with cooking modes, timer control, CLCD display, keypad input, to simulate the basic operation of a microwave oven.

## Hardware
- PIC16F877A
- CLCD
- Matrix Keypad

## Software
- MPLAB X IDE
- XC8 Compiler
- Embedded C
